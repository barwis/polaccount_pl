Colour System

Colour
Palette

Teal is the primary brand colour. Deep and authoritative, it signals trust and expertise without the coldness of corporate blue - essential for a business handling legal and financial matters across two markets.

Gold is used sparingly as an accent. It adds controlled warmth and references precision craftsmanship - not luxury. It elevates key moments without competing with the primary system.

Slate provides the neutral layer. Two tones - dark for text and structure, mid for secondary UI - keep interfaces calm, readable and grounded. No colour here is decorative.

Design Elements

UI
Elements

Minimal, functional components. Every state is deliberate. Colour is used to communicate role, not decoration. Teal signals primary action. Gold elevates a secondary path. Slate handles neutral structure.

Inter at controlled weights. Tight letter-spacing where precision matters. No rounded pill shapes - corners reflect the geometric logic of the mark.

All states shown. Never use colour alone to convey state.

Design Elements

Image
Treatment

All photography is processed through a CSS tint system to ensure brand consistency regardless of the source image. No filters are applied to the image element itself - a positioned overlay handles the effect.

Three utility classes cover all use cases. Apply the class to the image wrapper, not the img tag.

img-tint--background Teal cast + darken. Full-bleed hero sections with text overlay.
img-tint--darken Darken only. Blog thumbnails and cards.
img-tint--teal Light teal wash. Decorative images where clarity matters.

Apply to wrapper element, not img tag.
